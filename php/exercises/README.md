# PHP Exercises

One file per chapter. Work through them in order — each builds on the previous.

## How to use

1. Open the exercise file for the chapter you just finished reading
2. Fill in every `// TODO` block with your implementation
3. Run the file to verify your output
4. When done, paste the file content back to Claude for review

## How to run

Run any exercise from the repo root:

```bash
php php/exercises/chapter_NN.php
```

**Notes:**
- Chapters 28–33 use SQLite in-memory — no MySQL installation required
- Chapter 37 makes real HTTP calls and requires an internet connection

## How to get a review

Say: _"Please review my chapter N exercise"_ and paste or open the file.
Claude will check correctness, type safety, style, and suggest improvements.
