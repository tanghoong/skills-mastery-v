# TaskFlow Exercises — Progress Tracker

## How to Run

```bash
# From the express/ directory
npm install -g tsx

# Run a single exercise
tsx exercises/chapter_01.ts

# Run with watch mode
tsx watch exercises/chapter_01.ts
```

## Docker (required from Ch 1 onwards)

```bash
# Start postgres + redis
docker compose up -d

# Check logs
docker compose logs -f

# Stop
docker compose stop
```
