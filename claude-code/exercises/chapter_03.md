# Chapter 3 Drill — One-Liner vs. Five-Ingredient Prompt

The core skill of the whole course. Use the task from your Chapter 1 drill.

## Steps
1. Write **Prompt A**: the naive one-liner you'd normally type.
2. Write **Prompt B**: the five-ingredient version — Goal, Context, Scope, Example, Constraints (Ch. 3 §1).
3. Get the starting code into the same state twice (a scratch repo or a fresh `/clear`).
4. Run **Prompt A**, note the result. `/clear`. Run **Prompt B**, note the result.

## Reflection
- What did Claude get *wrong or have to guess* with Prompt A that Prompt B fixed?
- Which of the five ingredients made the biggest difference for *your* task?
- Was any ingredient unnecessary? (Matching effort to risk is part of the skill — Ch. 3 §7.)
- Which constraints did you repeat that should really live in `CLAUDE.md` (Ch. 5)?
