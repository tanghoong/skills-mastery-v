# Chapter 6 Drill — Context Hygiene

## Steps
1. **Clear-between-tasks:** Do task A, then *without clearing* start unrelated task B and notice any confusion. Then redo task B after a `/clear`. Compare.
2. **Compact:** Take one genuinely long single task; when it gets long, run `/compact` and confirm Claude continues coherently from the summary.
3. **Scratchpad:** Start a `PLAN.md` for a multi-step effort. `/clear`, then "read PLAN.md to get up to speed" and continue. Confirm the plan survived the reset.

## Reflection
- What symptom of a polluted context (Ch. 6 §2) did you actually observe in step 1?
- When did `/compact` feel right vs. when would `/clear` have been better?
- Why does `PLAN.md` survive a `/clear` when the conversation doesn't?
