# Chapter 2 Drill — Your First Session

Do this for real in a **scratch repo**.

## Steps
1. Install and authenticate Claude Code (Chapter 2 §1–2). Confirm with `claude --version`.
2. Launch `claude` from an empty scratch directory.
3. Run the two-step task from the chapter:
   - Create `hello.ts` with `greet(name)`.
   - Then: add `shout(name)` that reuses `greet()` instead of duplicating the string.
4. Run `/help` and read the list.
5. Mid-task (start any small request), press **Esc** once to feel the interrupt.

## Reflection
- When you asked for `shout`, you didn't re-describe `hello.ts`. How did Claude still know about it? (Tie this back to the context window, Ch. 1 §4.)
- List **three** slash commands from `/help` you didn't know existed.
- What happened when you pressed Esc? Why will that matter later?
