# Chapter 1 Drill — The Agent Mental Model

No coding. This drill builds the right mental model before you touch the tool.

## Part A — Spot the difference
Write down, for each, whether it's **autocomplete**, **chatbot**, or **agent** behavior:
1. You type `for (` and a loop body appears.
2. You paste a function into a web chat and it returns a fixed version to copy back.
3. You say "add validation to the signup form" and it finds the file, edits it, and runs the tests.

## Part B — The new-hire test
Pick a real task from your current project. Write:
1. The task in one sentence.
2. A bullet list of everything a brilliant-but-clueless day-one hire would need to *know* to do it well (conventions, file locations, gotchas, intent).

Keep Part B — you'll reuse it in Chapter 3.

## Reflection
- Which items in your Part B list could Claude *discover by reading the repo*, and which could it only know if *you told it*? (This distinction is the heart of prompting.)
- Why does an agent's autonomy make sloppy instructions more costly than with a chatbot?
