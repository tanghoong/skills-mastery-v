# Chapter 8 Drill — Interrupt, Redirect, Recover

Use a **scratch repo** so mistakes are free.

## Steps
1. **Interrupt:** Give a slightly ambiguous task and **Esc** the instant Claude does something unintended. Practice until the interrupt is reflex.
2. **Specific redirect:** After interrupting, redirect with a *concrete* correction ("reuse `formatDate` in `lib/date.ts` instead of a new util"). Confirm it lands on the next try.
3. **Git safety net:** `git commit` a checkpoint → ask for a risky change → `git restore` to throw it away cleanly.
4. **Restart:** Deliberately let a session get muddled, then practice `/clear` + a sharper prompt instead of nursing it.

## Reflection
- Compare a *vague* correction vs. a *specific* one you tried — how many rounds did each take?
- Recall your last frustrating AI session: which root cause from Ch. 8 §6 was it, and what `CLAUDE.md` line or habit would have prevented it?
- When did restarting beat redirecting?
