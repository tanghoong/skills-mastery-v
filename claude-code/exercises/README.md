# Claude Code Mastery — Exercises

One drill per chapter. Unlike a normal coding course, **these exercises are run *inside* a real Claude Code session.** The skill you're building is prompting and steering an agent, so the only way to practice is to do it live and observe.

## How to use

1. Read the chapter (`claude-code/chapter_NN_*.md`).
2. Open the matching drill (`claude-code/exercises/chapter_NN.md`).
3. Do the drill in a real Claude Code session (use a scratch/throwaway repo where noted, so mistakes are free).
4. Answer the **Reflection** questions at the bottom — that's where the learning sticks.
5. Update the tracker below.

## How to get a review

Say: _"Review my chapter N Claude Code drill"_ and paste your prompts, what Claude did, and your reflection answers. I'll give feedback on your prompting, scoping, and steering — the same way the TypeScript track reviews code.

## Setup

You need Claude Code installed and authenticated (Chapter 2). A scratch git repo is handy for the riskier drills:

```bash
mkdir cc-practice && cd cc-practice && git init
claude
```
